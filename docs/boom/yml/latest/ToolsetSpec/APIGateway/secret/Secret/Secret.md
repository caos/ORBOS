# Secret 
 

## Structure 
 

| Attribute  | Description                               | Default | Collection | Map  |
| ---------- | ----------------------------------------- | ------- | ---------- | ---  |
| encryption | Encryption algorithm used for the secret  |         |            |      |
| encoding   | Encoding algorithm used for the secret    |         |            |      |
| value      | Encrypted and encoded Value               |         |            |      |
